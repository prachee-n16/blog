---
title: JSON
---
*JavaScript Object Notation, hierarchical document schema to define objects with many attributes*

Best demonstrated with an example:
```json
{
	"fruits": [
		{
			"name": "apple",
			"colour": ["red"]
		},
		{
			"name": "orange",
			"colour": ["orange"],
			"taste": "interesting"
		}
	]
}
```
